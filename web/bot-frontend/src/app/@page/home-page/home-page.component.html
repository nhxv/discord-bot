About

Logo is a multi-functional Discord bot. It moderates your server, mutes spammers, checks the weather, verifies new members, denies users using VPNs access to your server, and much more.

Features

Anti-Spam: Auttaja removes unwanted spam and mutes the user who is posting it.

Anti-Advertisement: Auttaja will remove Discord invites from unauthorized users.

Moderation: Auttaja provides a plethora of different moderation commands

Gatekeeping: Auttaja prevents users that use a VPN from accessing your server
